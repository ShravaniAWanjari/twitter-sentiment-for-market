ModernBERT & Transformer
Benchmarking Suite

A modular, attribution-friendly benchmarking engine comparing ModernBERT, CryptoBERT, FinBERT, and DeBERTa-v3 on crypto sentiment analysis. Built to scale into Captum Integrated Gradients without refactoring.

Key Features

Unified Architecture

CryptoTransformerBase standardizes model wrappers with accessible embeddings, attention mechanisms, and forward passes that return logits, hidden states, and attention maps.

Multi-Model Support

Built-in support for ModernBERT, CryptoBERT, FinBERT, and DeBERTa-v3, all inheriting from a common base class for consistent behavior.

XAI Integration

Designed for explainability with exposed embedding layers and attention modules, ready for Captum Integrated Gradients analysis.

Getting Started

Installation & Setup

# Clone the repository git clone https://github.com/yourusername/crypto-transformer-benchmarking.git cd crypto-transformer-benchmarking # Install dependencies pip install -r requirements.txt

Data Preparation

# Prepare train/valid splits from the raw bitcoin dataset python experiments/prepare_bitcoin_sentiments.py \ --input bitcoin_sentiments_21_24.csv \ --output_dir dataset \ --neutral_band 0.05 \ --train_size 0.8

Training

# Train ModernBERT (Flash Attention enabled on GPU) python experiments/train_modernbert.py \ --train_path dataset/bitcoin_sent_train.csv \ --valid_path dataset/bitcoin_sent_valid.csv

Benchmarking

# Benchmark all models on the validation set python experiments/benchmark.py \ --valid_path dataset/bitcoin_sent_valid.csv \ --models modernbert cryptobert finbert deberta-v3 # Generate accuracy vs latency plots python performance_plots.py

XAI Capabilities

Embedding Access

Direct access to embedding_layer for Captum Integrated Gradients, enabling token-level attribution analysis.

Attention Inspection

Exposed attention_modules allow for attention head visualization and analysis across all layers.

Differentiable Forward

The forward_from_embeds() method supports attribution methods requiring differentiable inputs from embedding space.

Gradient Flow

Full gradient propagation through the model enables advanced explainability techniques and fine-grained analysis.
